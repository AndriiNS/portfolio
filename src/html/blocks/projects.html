About my projects

Usually, my projects are ordinary landing pages in which I use different approaches. For example, I often use JS, JQuerry, Swiper, Scrollreveal, and I also started learning animation with gsap. I also have no problems with the adaptive, I think I've got it down well.


In general, in every project I do, I come across something new and try to learn it so that it will be easier to do similar tasks in the future.In this project, for example, I learnt how to animate svg icons and how to work with gsap.
